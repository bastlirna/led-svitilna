## Součástky

* SMD Rezistor R1206 100R
* LED 5MM RED-ORANGE 4200
* Mikrospínač TD-03XG-T SMD
* Držák baterie do DPS pro CR2032 BC2001
* Fotocuprextit 200x150x1,5 jednovrstvý (pro 20 ks)
